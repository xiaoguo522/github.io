# 互联网医院基线项目介绍

# 首页

首页包含大部分常用功能的入口，有六个模块：

  • 顶部是一个搜索栏，点击可直接进入搜索医生或科室，快速挂号；
  • 接下来是线上就诊最常用的功能模块，功能分别是预约挂号、图文咨询、门诊缴费；
  • 紧跟着是最新消息资讯的轮播，点击可查看消息详情；
  • 首页中间就医指南模块主要是线上挂号咨询报到等功能入口集合，包含当日挂号、视频问诊、自助开单、复诊续方、住院充值、预约报到、用药咨询、护理咨询；
  • 我的健康模块主要是查询功能入口的集合，报告查询、我的预约、我的咨询、缴费记录、健康档案、体检报告、电子发票、住院清单；
  • 最下面是一个包含医院介绍、来院导航、健康资讯功能入口的模块；

# 预约挂号&当日挂号

功能介绍

预约挂号提供在线挂号功能，在不到医院的情况下可提前预约合适科室、时间段、医生的号源

# 1.专科选择

点击预约挂号功能入口，第一步是选择专科。进入专科选择页面首先弹出一个预约挂号的说明条款，阅读完方可点击阅读并同意，并进行专科选择。

# 2.选择医生、号源

医生选择如果是当日挂号则没有日期选择默认当天号源，如果是预约挂号有不限日期和选择指定日期两种方式：

  • 不限日期，点击挂号跳转号源选择页面
  • 选择日期，可以进行午别删选，如果该医生的号源有分时段的还需进行时段选择

选择号源

# 3.选择就诊人

选择就诊人并确认挂号信息

# 5.挂号成功

# 我的预约

我的预约在首页和个人中心都有单独入口，可以看到最近发预约挂号的记录，点击可查看预约记录详情 如遇挂号支付未成功的情况，在待付款一列可以取消预约，也可重新支付

# 预约报到

预约当天到达医院，可在已付款这一列，或者从首页的预约报到入口进入到预约报到列表进行线上报到排号

# 门诊缴费

功能介绍

首页有门诊缴费功能入口，进入查看所有待缴费的项目，选择需要缴费的项目进行缴费，其中线上问诊医生开的处方单可选择邮寄

# 1.选择缴费项目

门诊缴费列表可勾选要缴费的项目，点击项目可查看项目明细

# 2.确认缴费

如缴费项目中有线上问诊的处方单则可以选择窗口取药或者邮寄（其它情况线下操作）

# 3.缴费成功

# 缴费记录

首页我的健康模块有缴费记录入口，如果是邮寄可查看物流详情，点击单项缴费记录可查看缴费记录详情

# 图文咨询

功能介绍

图文咨询提供线上问诊服务，患者发送图片或文字来描述病情，医生接诊后可通过图文与患者沟通，为患者答疑解惑。

# 1.专科选择

点击图文咨询功能入口，第一步是选择专科。进入专科选择页面首先弹出一个云诊室的问诊须知，阅读完方可点击阅读并同意，再弹出问诊步骤介绍，滑动查看并确认，方可进行专科选择。

# 2. 选择医生，选择号源

# 3. 选择就诊人

选择就诊人，如果选中的就诊人是第一次在互联网医院就诊，会弹出初诊提示，需患者同意方可进入下一步填写病情

# 4. 填写病情

患者可填写病情描述和咨询目的，也可以上传图片，填写完点击提交咨询

# 5. 挂号支付

# 6. 挂号成功

# 我的咨询

我的咨询在首页和个人中心都有单独入口，可以看到最近挂图文咨询的记录，点击可查看沟通记录详情

如遇挂号支付未成功的情况，在待支付一列可以取消，也可重新支付

点击列表栏可进入沟通页面，可以给医生发送文字或图片消息，列表栏中可显示医生最新回复

医生接诊回复患者消息，患者可实时接收查看

# 结束问诊

结束问诊后，可查看处方，对医生进行评价

最新消息资讯的轮播，点击可查看消息详情

# 2. 选择医生，选择号源

# 3. 选择就诊人

选择就诊人，如果选中的就诊人是第一次在互联网医院就诊，会弹出初诊提示，需患者同意方可进入下一步填写病情

# 4. 填写病情

患者可填写病情描述和咨询目的，也可以上传图片，填写完点击提交咨询

# 5. 挂号支付

# 6. 挂号成功

# 视频问诊记录

视频问诊记录在个人中心有单独入口，可以看到最近挂视频问诊的记录，点击展示候诊信息

如遇挂号支付未成功的情况，在待支付一列可以取消，也可重新支付

就诊中列表栏点击可进入候诊页面，查看候诊信息

医生接诊呼叫患者，患者点击接入开始视频问诊沟通

# 结束问诊

结束问诊后，可查看处方，对医生进行评价

# 自助开单

功能介绍

自助开单功能入口在首页的就医指南模块，患者可自助线上下单医院开放的检查检验套餐项目，并进行缴费，方便快捷

# 1.选择开单项目

选择开单套餐进入自助开单项目明细页面，勾选需要的项目点击确认按钮会弹出一个自助开单就诊须知，阅读完确认进入支付挂号费页面

# 2.支付挂号费

点击确认支付挂号费会弹出一个新冠调查问卷，问卷填写正常后确认会进行自助开单的挂号费支付，支付完成会生成并查询本次开单的缴费项目并跳至门诊缴费确认缴费页面

# 3.支付项目费用

支付本次自助开单项目费用

# 用药咨询&护理咨询

功能介绍

  • 用药咨询属于图文咨询的一种，通过图文沟通的方式，指导患者正确的应用药物，可保证用药安全有效。
  • 护理咨询也属于图文咨询的一种，通过图文沟通的方式，对患者进行专业的护理指导，重点解决出行困难、需求迫切的护理问题，用专业知识帮助患者减轻痛苦。

# 1.选择医生

点击用药咨询功能入口，第一步是选择医生。进入页面首先弹出一个云诊室的问诊须知，阅读完方可点击阅读并同意，再弹出问诊步骤介绍，滑动查看并确认，方可选择医生。

# 2. 选择号源

# 3. 选择就诊人

# 4. 填写病情
